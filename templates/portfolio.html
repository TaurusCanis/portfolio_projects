Money Manager App

Github

I built my Money Manager App during the first few months of 2021. By logging expenses and transactions from all funding sources, the app helps users maintain a real-time understanding of their available funds. It essentially extends the idea of checkbook to include credit cards. The app was built using Django and deployed using Heroku.

Tauruscanis Rex Website and Store

Github

I built the Tauruscanis Rex website and store as part of a music project I worked on in summer and fall of 2020. The store uses the Braintree Payments API to process payments and Printful's API to submit and fulfill orders with their dropshipping service. tauruscanisrex.com is built on Django and hosted on Heroku.

Test Prep App

Github

In order to better help my students prepare for standardized tests, I built a test prep web app. Its functionality is fairly limited at this stage, though users can create an account, take practice tests, and review their results. The frontend makes use of Javacript to power the multiple-choice selection functions. It is built on Django and deployed using Heroku.

Tutor Tracker App

Github

At the beginning of 2020, I built an app to help me better organize and track appointments and billing for my tutoring business. The app allows users to create student and parent profiles, schedule sessions, record and email session recap notes, and generate invoices. It was built with Django and depolyed using Heroku.
